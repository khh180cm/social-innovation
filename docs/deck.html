참여자 상태를 빠르게 보고 · 판단 · 조치

합성 더미 데이터 300명 × 8주 기준 · 실데이터 검증 전

출석률·지급금액은 같은 축의 대리지표이고, 진짜 목표는 청년의 완주·역량입니다. 줄이려는 비효율은 '이탈을 너무 늦게 발견하는 것'.
S1 · 과제 소개 + 목표 위계

무엇을, 왜

  • 참가자 모집 → 8주(4주 정산 블록 2개). 지급은 출석률 한 숫자가 아니라 직무훈련·일경험 × 4주 블록 = 4셀별 산정
  • 목표 위계: 완주·역량(최상위) → 출석률(대리) → 적시 케어로 미지급 인원 최소화(운영)
  • 줄이려는 비효율 = 늦은 발견 (4주 정산 시점에야 이탈을 앎)
이미지: 프로그램 모집 공고
8~12주가 아니라 더미·코드 모두 8주. 지급은 4셀 합산이라 '출석률 하나 → 지급액 하나' 단일 도식이 아님.
S2 · 문제정의 (데이터로 교정)

어디에 집중할 것인가

  • 지급액은 출석률의 계단함수 — 상관을 "발견"한 게 아니라 규정상 정의다
  • 경계 한 칸이 금액을 점프시킨다
경계 통과직무훈련일경험
<50% → 50–60% (미지급→지급)+93,750+187,500
내부 경계 한 칸+18,750+37,500

실측: 인별 평균 <50% = 0명 · <70% = 74명(24.7%) · 4주 셀 단위 <50% = 61명/73셀

→ 집중 대상 = "50% 미만"이 아니라 정산 셀 단위로 경계(60–70%) 직하에서 흔들리는 인원. 상위군·심층 이탈자는 별도 트랙.

50% 절벽은 1인당 가치 5배지만 인별 평균으론 0명. 그래서 정산 셀 단위로, 사람이 있고 끌어올리기 싼 경계를 본다. 단위(인별 vs 셀)를 매번 명시.
S3 · AI 활용 설계 브레인스토밍런타임 규칙

같은 구간을 더 잘게

  • 물음: 같은 미달 구간도 다 같은 사람인가? 놓치는 맹점은?
  • 설계 AI: 더미를 주고 "맹점 찾아달라" → 같은 구간도 추세·회복 가능성·미지급 확률로 우선순위가 갈린다
  • 런타임: 그 통찰을 결정론적 규칙으로 구현(긴급도+priority 정렬). LLM 아님
이미지: 같은 구간 내 우선순위
AI를 설계 파트너로 썼다. 실행되는 분류 자체는 규칙(산수)이라 재현·감사가 된다. 화면 4등급은 출석률 밴드가 아니라 긴급도.
S4 · AI 활용 런타임 통계

미래를 내다보기

  • 업로드된 주만 관측, 남은 주는 예측 — Discounted Beta-Binomial + 코호트 prior, λ=0.8
  • 출력: 미지급 확률 · 기대 수당 · 80% 신용구간(점추정 아님)
  • 정직한 한계: 같은 블록 R²≈0.93, 다음 블록 R²≈0.034 — 출석만으론 선행 예측에 한계
이미지: 예측 뷰

→ 그래서 독립 신호가 필요하다 (S5로 연결)

예측은 LLM이 아니라 통계. 다음 블록은 출석만으론 거의 못 맞힘(R² 0.03). 강한 정확도조차 더미 5주차 하락에 기댄 것. '미래 예측 정확'을 단언하지 말 것.
S5 · AI 활용 런타임 LLM ★

빠르게 — 맞춤 처방

  • 물음: 알림톡 일괄 발송만으로 오를까? 출석률만으론 무엇을 왜 모른다
  • 수업 과정 신호 수집: 참가자(구글폼·알림톡 오픈율) + 평가자(교사·외부기관)
  • LLM claude-opus-4-8이 출석 예측 + 피드백을 묶어 주차별 조치·메시지 초안을 빠르게 생성 ← 유일한 런타임 AI
이미지: 맞춤 솔루션

⚠️ 예측은 출석으로, 피드백은 처방·근거 레이어(역할 분리). 더미 피드백 = 출석률의 거울 → 독립 효과는 실데이터 검증. 인과 단정 금지.

런타임 AI의 핵심. 300명을 일일이 못 보니 LLM이 무엇을 왜 할지 초안을 즉시 생성. 결정은 사람(human-in-the-loop). 효과 측정 루프는 향후 범위.
발표 내내 깔 정직성 단서
  1. 모든 수치는 합성 더미(300×8주), 실데이터 검증 전
  2. 피드백은 더미에선 출석률의 거울 → 독립 효과 입증 불가
  3. 5주차 하락은 생성기 산물일 수 있어 블록 간 예측은 달라질 수 있음
  4. "미지급 집중군" = 인별 0명 / 셀 단위 61명 — 단위 매번 명시
  5. 지급액↔출석률 "상관"은 발견이 아니라 정의(계단함수)
  6. 분류·우선순위·예측 = 규칙/통계, LLM = 조치·메시지 전용
정직한 한계 공개가 오히려 심사에서 신뢰를 키운다.
