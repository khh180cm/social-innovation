# Social Innovation — 미래내일 일경험 사업 운영 AI 판단 도구

> AX Arena 과제. 고용노동부 주관·대한상공회의소 운영 **미래내일 일경험 사업**의 운영 비효율을 줄이는
> AI 기반 판단 도구 프로토타입.

## 문제

운영자가 매주 다음 4가지를 빠르게 판단해야 한다.

1. 누가 어떤 상태인지
2. 누구를 우선 판단해야 하는지
3. 어떤 걸 우선 확인해야 하는지
4. 누구에게 어떤 안내·조치가 필요한지

## 목표

운영 데이터를 바탕으로 **빠르게 보고 → 판단하고 → 조치**할 수 있게 돕는다.
- 누가 위험한 상태인지 (출석률 하락·수당 미지급 위험 등)
- 누가 먼저 확인이 필요한지 (우선순위)
- 어떤 안내·조치가 필요한지 (AI 조치 추천)

## 제공 데이터

`participants.csv` — 가상 참여자 **300명** 8주치 더미 운영 데이터.

| 컬럼 | 설명 |
|------|------|
| `program_name` | 사업유형·운영기관 (`미래내일 일경험_A~E`, `직무훈련_A~E`, 각 30명) |
| `participant_name` | 참여자명 |
| `dummy_birth_gender_code` | 생년월일+성별코드 (더미) |
| `week_NN_job_training_attended_hours` / `_total_hours` | 주차별 직무훈련 출석/총 시간 |
| `week_NN_work_experience_attended_hours` / `_total_hours` | 주차별 일경험 출석/총 시간 |

## 수당 지급 규정 (판단 근거)

- 수당은 **4주마다** 지급, 출석률에 따라 차등.

| 구분 | 만점 지급액 | 90%+ | 70–90% | 60–70% | 50–60% | 50% 미만 |
|------|-----------|------|--------|--------|--------|---------|
| 직무훈련 수당 | 150,000원 | 150,000 | 131,250 | 112,500 | 93,750 | 미지급 |
| 일경험 수당 | 300,000원 | 300,000 | 262,500 | 225,000 | 187,500 | 미지급 |
